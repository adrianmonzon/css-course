Behind Vue

  • Diferencia con Javascript:
    • Javascript:
      • let message = 'Hello';
        let longMessage = message + ' world!';
        console.log(longMessage); // 'Hello world!'
        message = 'Hellooo';
        console.log(longMessage); // 'Hello world!'
        No hay reactividad
      • Proxy: El objeto Proxy permite crear un intermediario para otro objeto, el cual puede interceptar y redefinir operaciones fundamentales para dicho objeto. (ver aquí )
    • Vue: Funciona mediante proxies, que permiten rastrear las propiedades de los objetos para que cuando cambie el valor de estas, se actualice el nuevo valor, y otra serie de cosas (watchers, computadas, etc.)
  • refs: atributo para referenciar elementos html y manejar así cuando se requiera el DOM (this.$refs.nombreEstablecido)
  • Virtual DOM: Vue establece un DOM virtual en memoria y cada vez que algo cambia, compara el nuevo Virtual DOM con el anterior y modifica el DOM real sólo con los cambios que se han producido.
