Propiedades del contenedor Flexbox

Cambia los valores de cada propiedad y observa cómo se transforma el contenedor en tiempo real.

1. flex-direction

Define la dirección del eje principal: cómo se ordenan los ítems dentro del contenedor.

Valores de flex-direction:

  • row (por defecto): ítems en fila, de izquierda a derecha.
  • row-reverse: fila invertida, de derecha a izquierda.
  • column: ítems en columna, de arriba abajo.
  • column-reverse: columna invertida, de abajo arriba.
1
2
3
.contenedor {
  display: flex;
  flex-direction: row;  /* row | row-reverse | column | column-reverse */
}

2. flex-wrap

Controla si los ítems se mantienen en una única línea o pueden “saltar” a varias líneas.

Valores de flex-wrap:

  • nowrap (por defecto): una sola línea; los ítems se encogen para intentar caber.
  • wrap: permite varias líneas, que aparecen hacia abajo.
  • wrap-reverse: igual que wrap, pero las nuevas líneas aparecen hacia arriba.

.contenedor {
  display: flex;
  flex-wrap: nowrap;   /* nowrap | wrap | wrap-reverse */
}

3. flex-flow

Es la abreviatura de flex-direction + flex-wrap.

Valores de flex-flow (combinaciones):

  • row nowrap: fila en una sola línea.
  • row wrap: fila con salto a nuevas líneas hacia abajo.
  • column nowrap: columna en una sola “línea” vertical.
  • column wrap: columna donde las filas se apilan verticalmente.

.contenedor {
  display: flex;
  flex-flow: row nowrap;   /* <direction> <wrap> */
}

4. justify-content

Alinea y distribuye los ítems a lo largo del eje principal.

Valores de justify-content:

  • flex-start (por defecto): ítems agrupados al inicio del eje.
  • center: ítems centrados.
  • flex-end: ítems agrupados al final del eje.
  • space-between: primer ítem al inicio, último al final y espacio igual entre los demás.
  • space-around: cada ítem tiene espacio alrededor; los bordes exteriores reciben la mitad.
  • space-evenly: todos los espacios son exactamente iguales, incluyendo bordes.
A
B
C
.contenedor {
  display: flex;
  justify-content: flex-start;
  /* flex-start | center | flex-end
     space-between | space-around | space-evenly */
}

5. align-items

Alinea los ítems en el eje cruzado (perpendicular al principal).

Valores de align-items:

  • stretch (por defecto): los ítems se estiran para rellenar el eje cruzado.
  • flex-start: ítems pegados al inicio del eje cruzado.
  • center: ítems centrados en el eje cruzado.
  • flex-end: ítems pegados al final del eje cruzado.
  • baseline: alineados según la línea base del texto.

.contenedor {

6. align-content

Alinea el conjunto de filas o columnas de ítems cuando hay varias líneas (requiere flex-wrap).

Valores de align-content:

  • stretch (por defecto): las líneas se expanden para rellenar todo el eje cruzado.
  • flex-start: las líneas se agrupan al inicio del eje cruzado.
  • center: las líneas se agrupan en el centro.
  • flex-end: las líneas se agrupan al final del eje cruzado.
  • space-between: espacio solo entre líneas, primera y última pegadas a los bordes.
  • space-around: espacio alrededor de cada línea; los bordes tienen la mitad.
  • space-evenly: espacios iguales entre líneas y entre líneas y bordes.
