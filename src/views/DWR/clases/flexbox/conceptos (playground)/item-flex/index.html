Propiedades del Ítem Flexbox

Modifica las propiedades de cada ítem y observa cómo cambia su comportamiento dentro del contenedor flexible.

1. order

Cambia el orden visual de un ítem dentro del contenedor, sin alterar el orden real en el HTML.

Valores de order:

  • 0 (por defecto): posición original.
  • Números mayores: aparecen más tarde.
  • Números menores: aparecen antes.

2. flex-grow

Determina cuánto crece un ítem cuando hay espacio sobrante en el contenedor.

Valores de flex-grow:

  • 0: no crece (valor por defecto).
  • 1 o más: crece proporcionalmente.

3. flex-shrink

Indica cuánto puede encogerse un ítem cuando no cabe el contenido.

Valores:

  • 1 (por defecto): el ítem se encoge.
  • 0: NO se encoge (ideal para ítems “importantes”).
  • 2 o más: se encoge aún más que los demás.

4. flex-basis

Define el tamaño base inicial del ítem antes de aplicar crecimiento (flex-grow) o encogimiento (flex-shrink).

Valores:

  • auto: usa el contenido o el width.

5. flex (shorthand)

Abrevia flex-grow, flex-shrink y flex-basis.

Valores de ejemplo:

  • flex: 0 1 auto → valor por defecto. ("No crezcas aunque haya espacio sobrante, encógete cuando se estreche el contenedor y usa el tamaño de tu witdh original como punto de partida").
  • flex: 1 → igual a 1 1 0 (reparto igualado, opción más flexible). ("Crece si hay espacio sobrante en el contenedor, encógete cuando sea necesario y olvida tu tamaño inicial para ajustarte proporcionalmente").
  • flex: 0 0 auto → tamaño fijo. ("No crezcas aunque haya espacio sobrante, no te encojas cuando se estreche el contenedor y usa el tamaño de tu witdh original como punto de partida").
  • Los valores del selector son solo ejemplos.

6. align-self

Permite definir la alineación individual de un ítem en el eje cruzado, ignorando la alineación general del contenedor definida con align-items.

Valores de align-self:

  • auto: hereda la alineación del contenedor.
  • flex-start: al inicio del eje cruzado.
  • center: centrado en el eje cruzado.
  • flex-end: al final del eje cruzado.
  • stretch: ocupa todo el espacio disponible (cuando no se le ha asignado un height).
