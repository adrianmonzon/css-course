• Metadatos
    • charset: conjunto de caracteres que vamos a utilizar
      • Ej. UTF-8 (para las tildes, eñes, etc. entre otros caracteres)
  • Importar css: etiqueta link rel="stylesheet" type="text/css" href="styles.css"> en el head
  • Importar script: etiqueta script scr="script.js">:
    • justo antes de que cierre la etiqueta body
      • para que así haya cargado bien todo el contenido de la web cuando se llegue a la ejecución del script
    • si ese JS es crítico y tiene que ejecutarse antes de cargar el body, entonces irá en el head
